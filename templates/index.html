SDAcademy Description

Что можно узнать из курса

Практический курс по изучению современного языка программирования Python и фреймворка Django. Обучение будет проходить с уклоном в веб-разработку. Цель PyBursa научить начинающих программистов до уровня Junior Developer и после окончания курса все ученики могут претендовать на эту должность в любой интернет-компании. Весь курс построен на практике: преподаватели-практики, обучение на примере разработки реальных проектов, практические домашние задания. На курсе мы вас познакомим с языком Python, фреймворком Django, научим работать с базами данных, покажем, как проводить UnitTest, вместе сделаем работающий проект на Python/Django и освоим еще много полезного. После успешного окончания обучения всем предоставляется именной сертификат, который высылается почтой владельцу. Для нас это уже третья группа и многие из наших учеников успешно работают в сфере Python Development. Курс совершенствуется из года в год и становится еще более эффективным.

Плюсы и минусы языка Python

Положительные Отрицательные
Python - интерпретируемый язык программирования. это позволяет значительно упростить отладку программ Python - интерпретируемый язык программирования. Это обуславливает сравнительно низкую скорость выполнения
Огромное количество модулей, как входящих в стандартную поставку Python, так и сторонних. Отсутствие полноценной инкапсуляции - объявление имен с __ вначале не является ее полноценной заменой
Кроссплатформенность. Программа, написанная на Python, будет функционировать совершенно одинаково вне зависимости от того, в какой операционной системе она запущена. Отсутствие общей области видимости класса внутри методов класса (связано с переменными self)
